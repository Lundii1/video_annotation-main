# Video Annotation

## Installation
1. Install [Node.js](https://nodejs.org/en/download/).
2. `npm install` to install all the dependencies. 
3. `npm run build` to compile all to assets.

## Run
```bash
npm run dev
```
How to Use:

* After loading the video using arrow up and down you can choose the start and end of the segmentation and using arrow forward and back you can change the segmentation list after clicking on advance icon.
* You need to first choose the action for the segment and then use outcomes and person that you are labeling.
* You can find more details about how to use keyboard with the app in the help section of the app.
